Quake currently supports the basic workflow of uploading an RT-QuIC assay, making edits within individual wells, and visualizing immunofluorescence curves. These instructions will walk you through this workflow. Note that each experiment ("assay") is connected to a research team ("location"), plate template, and default sample record.



Edit Menu

The Edit menu contains tools for managing samples, plate templates, and samples. However, a "University of Wisconsin" location and a "Standard 96-well" plate template have already been created, so you likely will only need to ensure that a sample record is created to use with your assay.

Step 1. Go to Edit > Manage Samples, click "Create new sample", type in the details for your sample, and then click the "Submit new sample" button at the bottom. Note that if an appropriate sample record already exists, then you do not need to complete this step.



Upload Assay

Now you will create a new assay using a .csv file taken from the analyzer's output. The Upload Assay tool is also housed in the Edit menu.

Step 2. Go to Edit > Upload Assay. Ensure that the 96-well plate template, the UW location, and your default sample record is selected. (The default sample attached to the assay should be the most common sample across all wells. It will be associated with every well in the assay, but individual wells can then be updated in Well Edit after the upload is complete.) Enter an assay name and any other parameters that should apply to all wells in the assay. Press the Browse button at the bottom right to select the .csv file from your local file system. Press "Create assay" to upload. Note that this will take around 30-60 seconds to complete.



View Assay and Well Edit

Finally, you will be able to view the assay and make edits to individual wells.

Step 3. Go to Assay > View Assay, select your assay and all wells, and click Get Chart. You will see a grid of all IF curves.

Step 4. Left-click to select one or more wells, then right-click and go to Well Edit to change the properties of those specific wells.
